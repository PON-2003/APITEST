# Trash Classifier API

API สำหรับทำนายประเภทขยะจากภาพถ่ายหรือวิดีโอสตรีมมิ่งแบบเรียลไทม์

## วิธีใช้งาน

- `/predict` – ส่งภาพถ่ายและรับผลการทำนาย
- `/video_feed` – สตรีมวิดีโอพร้อมผลการทำนายแบบเรียลไทม์
